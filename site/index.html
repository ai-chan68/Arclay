核心工作流

不是黑盒 AI，而是确定性执行

EasyWork 拒绝盲目执行，每一步都在你的掌控之中。

📋

Planning

理解你的意图，生成详细的任务计划，明确交付物类型和执行步骤。

✅

Approval

审查 Agent 生成的计划，确认无误后批准执行。你始终拥有最终决策权。

⚡

Execution

在沙箱环境中执行计划，实时流式反馈，确保每一步都安全可控。

核心特性

为专业开发者打造的AI 工作台

四大核心能力，覆盖从代码生成到本地执行的完整开发生命周期。

🛠️

Harness 驱动

安全的本地命令执行，具备完善的错误恢复契约。每个错误路径都携带根因提示和重试指引。

🏗️

确定性执行流

快速开始

几分钟内启动

环境要求：Node.js ≥ 20，pnpm ≥ 9，Git，Rust (Stable)

Terminal
# 克隆仓库
$ git clone <repo-url> && cd EasyWork

# 安装依赖并启动
$ pnpm install && pnpm dev:all
命令 描述
pnpm dev:all 启动 Web 联调模式 (API + Frontend)
pnpm dev 启动 Tauri 桌面开发模式
pnpm build 全量构建项目
pnpm test 执行后端自动化测试
